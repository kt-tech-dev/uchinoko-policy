プライバシーポリシー 利用規約

🔒 プライバシーポリシー

最終更新日: 2026年3月24日

1. 収集する情報

本アプリでは、サービス提供のために以下の情報を収集します。

  • アカウント情報: メールアドレス（ログイン認証に使用）
  • ペット情報: ペットの名前・種類・品種・誕生日・体重などの基本情報
  • 記録データ: 食事・散歩・トイレ・体調などの日々の記録、体重ログ、ワクチン・投薬・受診記録、日記、写真
  • 支出データ: ペット関連の支出記録

2. 情報の利用目的

収集した情報は以下の目的にのみ使用します。

  • アプリの機能提供およびサービスの維持・改善
  • ユーザー認証およびアカウント管理
  • 家族共有機能によるデータの共有
  • 通知リマインダーの送信

3. データの保存

ユーザーデータは Supabase（AWS インフラストラクチャ上）に暗号化して保存されます。写真データは Supabase Storage に保存されます。通知設定は端末のローカルストレージに保存されます。

4. 第三者への提供

ユーザーの個人情報を第三者に販売・貸与することはありません。

ただし、明細書読み取り機能をご利用の場合、アップロードされた画像は Google Gemini API に送信され、テキスト抽出処理が行われます。画像は処理後に保存されません。

5. データの削除

アカウントを削除すると、関連するすべてのデータ（ペット情報・記録・写真等）が完全に削除されます。削除のご要望はアプリ内の設定、またはお問い合わせ先までご連絡ください。

6. お問い合わせ

プライバシーに関するご質問は、uchinokolog@gmail.com までご連絡ください。

📄 利用規約

最終更新日: 2026年3月24日

1. サービスの概要

うちの子けんこう手帳（以下「本アプリ」）は、ペットの日々の記録・健康管理を支援するアプリケーションです。

2. アカウント

本アプリの利用にはアカウント登録が必要です。ユーザーは正確な情報を提供し、アカウントの管理に責任を持つものとします。

3. 禁止事項

  • 法令に違反する行為
  • 他のユーザーへの迷惑行為
  • サービスの運営を妨害する行為
  • 不正アクセスまたはその試み

4. 免責事項

本アプリはペットの健康管理を補助するツールであり、獣医療の代替ではありません。ペットの健康に関する判断は、必ず獣医師にご相談ください。

明細書読み取り機能（OCR）はAIによる自動処理のため、読み取り結果の正確性を保証するものではありません。必ず原本と照合してください。

本アプリの利用により生じた損害について、開発者は一切の責任を負いません。

5. サービスの変更・終了

開発者は、事前の通知なくサービスの内容を変更、または提供を終了する場合があります。

6. 規約の変更

本規約は必要に応じて変更される場合があります。重要な変更がある場合は、アプリ内で通知します。
