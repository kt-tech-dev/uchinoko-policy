プライバシーポリシー

最終更新日: 2026年3月24日

1. 収集する情報

本アプリでは、サービス提供のために以下の情報を収集します。

アカウント情報: メールアドレス（ログイン認証に使用）

ペット情報: ペットの名前、種類、品種、誕生日、体重などの基本情報

記録データ: 食事・散歩・トイレ・体調などの日々の記録、体重ログ、ワクチン・投薬・受診記録、日記、写真

支出データ: ペット関連の支出記録

2. 情報の利用目的

収集した情報は以下の目的にのみ使用します。

  • アプリの機能提供およびサービスの維持・改善
  • ユーザー認証およびアカウント管理
  • 家族共有機能によるデータの共有
  • 通知リマインダーの送信

3. データの保存

ユーザーデータは Supabase（AWS インフラストラクチャ上）に暗号化して保存されます。写真データは Supabase Storage に保存されます。通知設定は端末のローカルストレージに保存されます。

4. 第三者への提供

ユーザーの個人情報を第三者に販売・貸与することはありません。

ただし、明細書読み取り機能をご利用の場合、アップロードされた画像は Google Gemini API に送信され、テキスト抽出処理が行われます。画像は処理後に保存されません。

AI健康相談機能をご利用の場合、入力されたメッセージおよびペットの健康データは Google Gemini API に送信されます。

5. データの削除

アカウントを削除すると、関連するすべてのデータ（ペット情報、記録、写真等）が完全に削除されます。データ削除のご要望はアプリ内の設定またはお問い合わせ先までご連絡ください。
